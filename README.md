# Geometry-Solver-CPP
Ein kleines C++ Tool zur Berechnung von Schnittpunkten zwischen Geraden und Parabeln.

## Features
- **Geraden-Klasse:** Erstellung über m/b oder zwei Punkte.
- **Schnittpunkt-Berechnung:** Findet Schnittpunkte zwischen zwei Geraden oder einer Gerade und einer Parabel.
- **Punkt-Check:** Prüft mit Toleranzbereich, ob ein Punkt auf einer Geraden liegt.

## Installation
Einfach die Header-Dateien einbinden oder die `main.cpp` kompilieren:
`g++ main.cpp -o geometrie_test`
